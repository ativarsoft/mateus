Server location and SEO

by Mateus de Lima Oliveira

Google and other search engines operate according to the laws depending on the server is. That also applies to search results.

If you have a perfectly legal and nice website but the goverment where the server is represses and censors people, move the server and the DNS to a country or location where it is legal and the goverment in that location is not known for such practices.

Make sure to move all domains including subdomains.
